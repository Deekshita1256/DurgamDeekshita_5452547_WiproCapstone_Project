Make My Trip Selenium Capstone Project
This project automates Make My Trip Cruise booking scenarios with Selenium, Behave BDD, and Allure reporting.

Setup
pip install -r requirements.txt

Run Tests
pytest tests/


Make My Trip Behave BDD Capstone Project
This project automates Make My Trip Cruise booking scenarios with Selenium, Behave BDD, and Allure reporting.

Setup
pip install -r requirements.txt

Run Tests
behave

To clean old reports, run Behave, and generate the Allure report:
python runtests.py
